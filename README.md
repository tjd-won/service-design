# 서비스·경험디자인 기사 · 실기 올인원

서비스·경험디자인기사 **실기** 대비용 자기주도 학습 도구입니다.
개념 이해 → 실전 문제 풀이 → 오답 암기를 **하나의 웹앱**에서 진행합니다.

🔗 **바로가기:** https://tjd-won.github.io/service-design/
*(GitHub Pages 배포 후 실제 주소로 교체하세요)*

---

## 구성

| 탭 | 내용 |
|---|---|
| 🏠 현황 | D-day, 개념·문제·암기 진도 한눈에 보기 |
| 📚 개념 대시보드 | 핵심 개념 정리 · 검색 · 이해/복습 체크 |
| ✍️ 실전 문제 | 문제 풀이 · 자가 채점(정답/부분/오답) · 오답을 암기 드릴로 전송 |
| ⚡ 암기 드릴 | 카드 뒤집기 방식 반복 암기 |

## 사용법

1. 위 주소를 **모바일 Safari / PC 브라우저**로 엽니다.
2. iPhone·iPad는 **공유(⬆️) → 홈 화면에 추가** 하면 앱 아이콘처럼 쓸 수 있습니다.
3. 학습하면서 체크·채점하면 진도가 자동 저장됩니다.

> ⚠️ 로컬 HTML 파일을 아이폰 **‘파일’ 앱**에서 직접 열면 자바스크립트가 실행되지 않아 화면이 비어 보입니다. 반드시 **웹 주소(Safari)** 로 여세요.

## 진도 저장

- 진도는 브라우저 **localStorage**에 저장됩니다 → **기기별로 따로** 저장되고, 캐시 삭제 시 사라집니다.
- 기기 간 이동·백업은 홈 화면의 **⬇️ 진도 백업 / ⬆️ 진도 불러오기** 기능을 사용하세요.

## 기술 메모

- **단일 HTML 파일**(`index.html`), 외부 의존성 없음(인라인 CSS/JS).
- 4개 화면은 `iframe srcdoc`로 격리 로드(각 화면의 스크립트 충돌 방지).
- iOS 대응: 뷰포트 높이 픽셀 고정, JS 미실행 환경 안내(noscript), 검색 색인 차단(`noindex`).

## 수정 / 업데이트

`index.html`을 새로 올리거나 저장소에서 직접 편집하고 커밋하면, GitHub Pages가 1~2분 내 자동 재배포합니다.
